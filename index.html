DISPLAY: ES LO QUE MOSTRAMOS EN NUESTRA PANTALLA.

Display None

Este tipo de display es especial porque nos permite dejar de visualizar contenido dentro de mi sitio.

Cuando ocupamos un display none, las etiquetas dejan de verse pero el código sigue ahí.

Display Block

El elemento genera un cuadro de elemento de bloque.

Significa que mi bloque recuerda que todo en HTML es un bloque y ocupa todo el espacio disponible de izquierda a derecha; o dicho de otra forma, todo el renglón dentro del componente padre.

Display Inline

El elemento genera uno o más cuadros de elemento en línea.

Significa que el contenedor o bloque se adapta al ancho de su contenido hijo.

Display Inline-Block

Si tu le pones tamaño a un display block este funciona pero, sigue ocupando todo el renglón.

Esto puede ser problemático si yo quiero controlar el tamaño de mi componente y que el contenido esté en la misma línea, para eso existe el display inline block.
